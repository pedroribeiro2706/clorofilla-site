# Clorofilla - Site Institucional

Site institucional da Clorofilla - Soluções Ambientais com Propósito.

## Estrutura do Projeto

O site utiliza uma estrutura modular com carregamento dinâmico:

- `index.html`: Contém placeholders para cada seção do site
- `script.js`: Controla o scroll horizontal/vertical, efeitos parallax e menu
- `style.css`: Estilos globais, primeira seção (hero) e menu
- `js/loader.js`: Gerencia o carregamento dinâmico das seções via fetch()
- `secoes/`: Contém os arquivos HTML de cada seção do site
- `css/`: Estilos específicos para cada seção
- `js/`: Scripts específicos para cada seção
- `assets/`: Imagens e recursos visuais
- `Knowledge/`: Documentação e informações sobre o projeto

## Características

- Carregamento dinâmico de seções via fetch()
- Animações com GSAP
- Efeitos parallax
- Design responsivo
- Estrutura modular para facilitar manutenção
